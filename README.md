## About The Project

<!--[![Product Name Screen Shot][product-screenshot]](https://example.com)-->

Its time to take your competitive programming hobby seriously! Keep track and document the problems you solved and attempted. Analyze the generated metrics to find out your shortcommings and strengths to further improve your problem solving skills!

But, but ... why choose AptSolve over an Excel sheet? Here's a few reasons why:
* A beautiful and responsive web application, a stark contrast to the plain and static Excel sheets...
* Generated visual metrics that beautify and expresses your data.
* Powerful search functionality lets you find the problem set you want fast and easily.
